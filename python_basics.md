
### Logical operators

| A     | NOT A |
|-------|-------|
| false | true  |
| true  | false |

| A     | B     | A AND B | A OR B |
|-------|-------|---------|--------|
| false | false | false   | false  |
| false | true  | false   | true   |
| true  | false | false   | true   |
| true  | true  | true    | true   |
