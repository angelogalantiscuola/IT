# Python
## Execute files
### Python script - Run a Python file as a script.
### Interactive Python - Use Python in an interactive shell.
## Indentation
### Code blocks - Blocks of code are denoted by indentation.
### Consistency - Consistent indentation is crucial in Python.
## I/O
### print() - Output data to the console.
#### strings - Print text data.
#### numbers - Print numerical data.
#### variables - Print the value of variables.
### input() - Read data from the console.
#### strings - Read text data.
#### data type conversion - Convert the input data to the required data type.
## Variables
### assignment (=) - Assign a value to a variable.
### variable names - Names given to variables for identification.
### global and local variables - Variables defined in the global scope and within a function respectively.
## Comments
### single-line comments - Comments that span only one line.
### multi-line comments - Comments that span multiple lines.
## Data types
### type() - Get the data type of a variable.
### Casting
#### str() - Convert a variable to a string.
#### int() - Convert a variable to an integer.
#### float() - Convert a variable to a float.
### str
#### slicing - Extract parts of a string.
#### len() - Get the length of a string.
#### concatenation - Combine two strings.
#### f-strings - Format strings in Python.
#### escape characters - Special characters in strings.
### bool
#### True - Boolean True value.
#### False - Boolean False value.
### int
#### arithmetic operations - Perform calculations with integers.
#### comparison - Compare integers.
### float
#### arithmetic operations - Perform calculations with floats.
#### comparison - Compare floats.
### list
#### len() - Get the number of items in a list.
#### append() - Add an item to the end of a list.
#### remove() - Remove an item from a list.
#### insert() - Add an item at a specific position in a list.
#### pop() - Remove an item from a specific position in a list.
#### index() - Find the index of an item in a list.
#### count() - Count the number of times an item appears in a list.
#### sort() - Sort the items in a list.
#### reverse() - Reverse the order of items in a list.
### dict
#### values() - Get all the values in a dictionary.
#### keys() - Get all the keys in a dictionary.
#### items() - Get all the key-value pairs in a dictionary.
#### get() - Get the value of a specific key in a dictionary.
#### update() - Update the value of a specific key in a dictionary.
#### pop() - Remove an item from a dictionary.
#### clear() - Remove all items from a dictionary.
## Operators
### arithmetic
#### addition (+) - Add two numbers.
#### subtraction (-) - Subtract one number from another.
#### multiplication (*) - Multiply two numbers.
#### division (/) - Divide one number by another.
#### modulo (%) - Get the remainder of a division.
#### exponent (**) - Raise a number to the power of another number.
### assignment
#### simple assignment (=) - Assign a value to a variable.
#### assignment with operator (+=, -=, *=, /=) - Perform an operation and assign the result to a variable.
### comparison
#### equal to (==) - Check if two values are equal.
#### not equal to (!=) - Check if two values are not equal.
#### less than (<) - Check if one value is less than another.
#### greater than (>) - Check if one value is greater than another.
#### less than or equal to (<=) - Check if one value is less than or equal to another.
#### greater than or equal to (>=) - Check if one value is greater than or equal to another.
### logical (and, or, not) - Logical operators.
## if...else
### if - Execute a block of code if a condition is true.
### else - Execute a block of code if the condition in the if statement is false.
### elif - Execute a block of code if the previous conditions were not true.
## Loops
### for - Execute a block of code for each item in a sequence.
#### for .. in [] - Loop through each item in a list.
#### for .. in {} - Loop through each key-value pair in a dictionary.
#### for .. in range() - Loop through a sequence of numbers.
### while - Execute a block of code while a condition is true.
#### while with condition - Execute a block of code while a condition is true.
#### break - Exit the loop prematurely.
#### continue - Skip the current iteration and continue with the next one.
## Functions
### Function definition - Define a new function.
### Parameters - Inputs to a function.
### Return values - Outputs of a function.
## Modules
### import - Import a module.
### from ... import ... - Import a specific function or variable from a module.
### import ... as ... - Import a module under a different name.
### from ... import * - Import all functions and variables from a module.
